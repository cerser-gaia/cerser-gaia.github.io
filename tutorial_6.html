Exploring MultiSpec



Exporting Image Data: Part 2

In this tutorial we are going to cover some of the functionality of MultiSpec.



  • Following the previous tutorial, we will be getting the histogram data of the same image. To download the image use this link.

  • Histogram data is used for determining the gray scale regime for a display or for listing and graphing.

  • Begin by clicking Histogram Image from the Processor tab.

  • This will open a new window.

  • In this window you can see 6 input boxes. These boxes are use to change the section of the image that will be used.

  • For now we are going to leave the values at their default values. This means we will be using the entire image.

  • In the bottom left corner of the box you will see a checkbox labeled List Histogram. Make sure this box is checked.

  • Click the OK button. The output data should now appear in the already open text box.

  • Observe the output data.
